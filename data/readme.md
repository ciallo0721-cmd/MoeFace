
# MoeFace 人脸识别数据集

本数据集最初是为 **MoeFace** 的人脸识别准备的，但里面的很多图片也适合当作壁纸、背景或风格迁移素材。  
所有图片均来自公开网络，使用特定关键词搜索获得，**不包含任何真人隐私照片**。

> 每个文件夹代表一个独立角色，内部存放该角色的面部图像。

## 特性

- **完全免费**：无任何商业使用限制
- **每日更新**：持续补充新角色和新图片
- **人工审查**：确保图片内容与文件夹标注的角色一致，避免误混入其他人物
- **开箱即用**：目录结构清晰，可直接用于人脸识别模型的训练或微调

## 使用方式

### 直接下载

### 在模型中使用
```python
# 示例：使用 PyTorch 的 ImageFolder 加载
from torchvision.datasets import ImageFolder

dataset = ImageFolder("data/")
print(dataset.classes)   # 即所有角色名称列表
```

### 作为背景图使用
部分图片也可作为风格化背景素材，请自行筛选。
