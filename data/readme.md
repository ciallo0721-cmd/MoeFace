# 爬虫搜索关键词使用说明

## 避免不相关图片的方法

- 使用减号 `-` 排除特定词，例如：
  - `忆雨 -warma`
  - `monika - monica`
  - `永雏塔菲 -黑桃影`

- 使用空格表示搜索关键词必须同时包含两个词（不要求顺序），例如：
  - `monika ddlc`

- 使用加号 `+` 表示搜索关键词包含两个词（不要求顺序），例如：
  - `monika + ddlc`

## 限定网站搜索

在搜索关键词后加上 `site 网址`，可以只搜索特定网站上的图片，例如：
- `site twitter.com monika ddlc` — 只搜索 Twitter/X 上关于 monika ddlc 的图片
