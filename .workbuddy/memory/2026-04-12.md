## MoeFace CLI 模式扩展

应用户要求，为 recognize.py 新增 CLI（终端图形化）模式：

**入口逻辑**：
- `python recognize.py --mode gui`（默认）→ 启动 Tkinter GUI
- `python recognize.py --mode cli` → 启动终端彩色 UI
- 不加 `--mode` 参数默认进入 GUI

**新增参数**：
- `--mode cli/gui`：选择模式
- `--source`：文件路径
- `--camera`：摄像头模式
- `--output`：输出路径
- `--db-name`：特征库名称
- `--threshold`：识别阈值
- `--skip-frames`：跳帧数
- `--min-neighbors`：检测灵敏度
- `--rebuild`：强制重建特征库
- `--list`：列出所有可用特征库（不加载模型）

**CLI 特性**：
- ASCII 艺术 Banner（启动时显示）
- ANSI 彩色日志（info/warn/error/done 四级）
- 视频处理实时进度条（同一行覆盖）
- 摄像头按 Q 退出（兼容 CV2 waitKey）
- 识别报告（角色分布 + 柱状图）
- Ctrl+C 安全中断
