# 2026-04-13 日常记录

## MoeFace 项目更新

### 特征库加载优化
- 重构 `build_database` 函数：支持 `progress_fn` 回调、多线程并行处理
- `get_or_build_database` 增加 `progress_fn` 参数
- GUI 新增进度条组件（ttk.Progressbar + 剩余时间估算）
- 多线程配置：`min(cpu_count, 8, total_images)` 防止资源耗尽
- 每处理 5 张图更新一次进度，减少 UI 刷新开销

### 关于 animeface
- 当前已使用 `lbpcascade_animeface.xml`（动漫专用级联分类器）
- 如果要加 OpenCV 内置的 `haarcascade_frontalface_default.xml` 做双检测，需在 `extract_features_from_image` 和 `process_image_file`/`process_video_file` 中同时调用两个 cascade，合并结果
